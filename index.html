Czym jest Git?

Git to system kontroli wersji, który pozwala na śledzenie zmian w plikach w czasie. Git umożliwia współpracę wielu osób nad jednym projektem, zarządzanie wersjami plików, a także odzyskiwanie poprzednich wersji kodu.

Podstawowe komendy Git

Git oferuje szereg komend, które pomagają zarządzać repozytoriami i wersjami kodu. Oto kilka z nich:

  • git init - Inicjalizuje nowe repozytorium Git w bieżącym katalogu.
  • git add plik - Dodaje zmiany w pliku do obszaru staging (czyli do przygotowania do commitowania).
  • git commit -m "wiadomość" - Tworzy nowy commit, zapisując zmiany w repozytorium.
  • git status - Sprawdza status repozytorium, pokazując zmiany w plikach.
  • git log - Wyświetla historię commitów w repozytorium.
